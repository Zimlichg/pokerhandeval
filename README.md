# Poker Hand Evaluator

This project is a Texas Hold'em poker hand evaluator and simulator built in Java. It was created as a 1-day learning project to practice Object-Oriented Programming (OOP) fundamentals and pick up some simple Java concepts, using a topic (poker) that I genuinely enjoy. 

## Features
- A standard 52-card deck with shuffling functionality.
- Hand evaluation logic for standard poker hands (from High Card up to Royal Flush).
- A simulator that deals 5 cards to multiple players and determines the winner of a poker round.

## How to Run

There are a couple of very simple ways to play a simulated hand:

### 1. The Simple Script (run.bat)
I have included a `run.bat` script that handles everything behind the scenes. 
From your terminal, just type:
```powershell
.\run.bat
```
*(Or, you can simply double-click the `run.bat` file in your File Explorer!)*

### 2. The IDE Button (VSCode)
Since you are using VSCode with Java extensions, you can open `src/main/java/com/poker/Main.java` and simply click the **"Run"** button that appears directly above your `main` method (line 7).
